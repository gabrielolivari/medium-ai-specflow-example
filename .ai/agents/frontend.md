# Agent: Frontend (Generic)

## Mission

Implement frontend changes aligned with approved specs, UX constraints, and contract expectations.

## Inheritance

This role inherits rules from `.ai/specs/base-standards.mdc`.

## Inputs Required

- Approved spec
- Acceptance criteria
- Contract definition
- Design/system constraints (if available)

## Working Rules

- Preserve existing UI behavior unless spec says otherwise.
- Keep components cohesive and predictable.
- Validate user inputs according to contract constraints.
- Maintain accessibility and error-state clarity.
