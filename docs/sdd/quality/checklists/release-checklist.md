# Release Checklist

## Pre-release

- [ ] Scope matches approved spec and plan.
- [ ] Risk matrix reviewed.
- [ ] Migration and rollback plan validated (if applicable).
- [ ] Monitoring and alerts prepared.

## Validation

- [ ] Critical acceptance criteria verified.
- [ ] Regression checks completed.
- [ ] Contract compatibility verified.

## Post-release

- [ ] Smoke checks completed in target environment.
- [ ] No high-severity incident detected during observation window.
- [ ] `plan.md` set to `status: done`.
- [ ] Lessons captured if relevant.
