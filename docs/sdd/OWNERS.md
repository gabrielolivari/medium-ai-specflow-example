# Owners - SDD Domain

## Teams

- Tech Leads: `@your-org/tech-leads`
- QA: `@your-org/qa`
- Developers: `@your-org/devs`
- AI Engineering: `@your-org/ai`

## Scope

This ownership applies to everything under `docs/sdd/`.

## Responsibility Split

- **tech-leads**: architecture direction, approvals for significant changes.
- **qa**: quality gates, test strategy, release readiness checks.
- **devs**: spec implementation alignment and plan execution.
- **ai**: agent behavior, prompt quality, and automation governance.

## Review Policy

- At least one `tech-leads` approval for decision-level changes.
- At least one `qa` approval for changes under `quality/`.
- At least one `ai` approval for changes under `.ai/` and framework rules.
